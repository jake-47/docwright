# docwright

Live at <https://jake-47.github.io/docwright>.

Personal blog built with [Zola](https://www.getzola.org/); styling borrows from [Giallo](https://github.com/getzola/giallo).

Local preview (after installing Zola — see the guide):

    ./serve      # http://127.0.0.1:1111, live reload
    ./build      # build into ./public

Deploys to github (<https://github.com/jake-47/docwright>) automatically on every push to `main`.

Setup, hosting, and customisation: zola-blog-guide.md.
