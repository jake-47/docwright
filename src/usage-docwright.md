# Usage
<p class="mdb-subtitle">Guide to using the docwright knowledge-base</p>

## Navigation

There are several methods for navigating through the pages of the knowledge base.
The **menu button** (three horizontal bars) at the top-left of the page to open and close the sidebar.
The **sidebar** lists all the sections and topics of the knowledge base.
Click any topic to load that page.

The keyboard shortcut <kbd>b</kbd> opens and closes the sidebar.

The <kbd><</kbd> left and right <kbd>></kbd> arrow keys moves you to the previous and next page; the big arrow buttons at the sides of the page do the same thing.

The <kbd>∧</kbd> up and down <kbd>v</kbd> arrow keys scroll up and down a page.

The <kbd>PgUp</kbd> and <kbd>PgDn</kbd> jump high-level headings; <kbd>Spacebar</kbd> and <kbd>Shift + Spacebar</kbd> keys do the same thing.

> [!NOTE]
> If the window is too narrow, particularly on mobile displays, sidebar will not automatically appear.
In that case, use the menu button to open and close the sidebar.

## Top menu bar

| Icon | Description |
|------|-------------|
| <i class="fas fa-bars"></i> | Opens and closes the sidebar. |
| <i class="fas fa-magnifying-glass"></i> | Opens the search bar. |
| <i class="fas fa-print"></i> | Print the entire knowledge base. Press <kbd>Ctrl + p</kbd> to print only the current page.|
| <i class="fab fa-github"></i> | Opens a link to the provider hosting the source code of the knowledge base. |
| <i class="fas fa-pencil"></i> | Opens the page to directly edit the source of the page you are currently reading. |
| <i class="fa fa-copy"></i> | Copies the code in the code block to your local clipboard, so you can paste it into another application. |
|  | Clicking any where on the menu bar, other than the icons, will take you to the top of the page. |


> [!NOTE]
> Changing your system's theme (light or dark) will automatically change this site's theme.

## Search

You can use the search icon <i class="fas fa-magnifying-glass"></i> in the menu bar or the easier keyboard shortcuts <kbd>/</kbd> or <kbd>S</kbd>.
Enter keywords to see matching topics and headings in real time.
The <kbd>∧</kbd> and <kbd>v</kbd> arrow keys help you select a result, and <kbd>Enter</kbd> opens that section.
<kbd>Esc</kbd> closes the search bar.
